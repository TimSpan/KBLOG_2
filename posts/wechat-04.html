2.1сђЂу╗ёС╗ХТаит╝Јжџћуд╗ #

  • у╗ёС╗Х A уџёТаит╝ЈСИЇС╝џтй▒тЊЇу╗ёС╗Х C уџёТаит╝Ј
  • у╗ёС╗Х A уџёТаит╝ЈСИЇС╝џтй▒тЊЇт░ЈуеІт║ЈжАхжЮбуџёТаит╝Ј
  • т░ЈуеІт║ЈжАхжЮбуџёТаит╝ЈСИЇС╝џтй▒тЊЇу╗ёС╗Х A тњї C уџёТаит╝Ј

  • тЈфТюЅ class жђЅТІЕтЎеС╝џТюЅТаит╝Јжџћуд╗ТЋѕТъю№╝ї id жђЅТІЕтЎесђЂт▒ъТђДжђЅТІЕтЎесђЂТаЄуГЙжђЅТІЕтЎеСИЇтЈЌТаит╝Јжџћуд╗уџётй▒ тЊЇ

2.3сђЂС┐«Тћ╣у╗ёС╗ХуџёТаит╝Јжџћуд╗жђЅжА╣ #
